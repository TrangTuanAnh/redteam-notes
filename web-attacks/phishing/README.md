# Phishing

[← Web Attacks](../README.md)

Phishing là kỹ thuật tấn công phi kỹ thuật số (social engineering) nhắm vào con người thay vì hệ thống — giả mạo danh tính đáng tin cậy để lừa nạn nhân cung cấp credential, cài malware, hoặc thực hiện hành động ngoài ý muốn.
